# WeChat Download API 使用指南

## 快速开始

### 1. 启动服务

**双击 `start.bat`**，脚本会自动完成所有步骤。

- 已安装 Python → 自动装依赖 + 启动
- 未安装 Python → 自动弹出下载页面，安装后重新双击即可

### 2. 扫码登录

服务启动后，用浏览器打开：**http://localhost:5001/static/login.html**

用微信扫二维码完成登录。登录凭证自动保存，有效期约 **4 天**。

### 3. 空闲自动关闭

服务支持**无请求自动退出**：每次请求完成后开始计时，**20 分钟没有任何新请求，进程自动关闭**。

- 不需要手动执行 `stop.sh`，用完不管它自己会停
- 如果需要立即停止，双击 `stop.bat` 即可
- 如需修改超时时间，在 `.env` 中添加：
  ```bash
  IDLE_TIMEOUT_SECONDS=1200   # 20分钟（默认）
  IDLE_TIMEOUT_SECONDS=3600   # 改为1小时
  IDLE_TIMEOUT_SECONDS=0      # 禁用自动关闭
  ```

## 使用方式：人用 vs Agent 用

这个服务同时支持两种使用方式，接口完全一致。

### 方式一：人用 — 浏览器操作

| 地址 | 用途 |
|------|------|
| http://localhost:5001/static/admin.html | **管理面板** — 查看登录状态、API 文档、快捷操作 |
| http://localhost:5001/static/login.html | **扫码登录** — 首次使用或凭证过期时用 |
| http://localhost:5001/static/rss.html | **RSS 订阅管理** — 搜索公众号、一键订阅、复制 RSS 地址 |
| http://localhost:5001/api/docs | **Swagger 文档** — 在线调试所有接口（适合手动测试） |

#### 举例：查询某公众号昨天发了什么

1. 打开 http://localhost:5001/static/admin.html
2. 找到"搜索公众号"区块，输入公众号名称，复制 FakeID
3. 找到"获取文章列表"区块，填入 FakeID，复制结果到本地过滤

#### 举例：提取某篇文章的正文内容

1. 打开 http://localhost:5001/api/docs
2. 找到 `POST /api/article`，填入文章链接，点击 Execute
3. 直接在页面上看到标题、正文、图片列表

---

### 方式二：Agent 用 — API 调用

Agent 直接调用 REST 接口，不需要浏览器。

#### 功能一：文章链接 → 正文内容

```bash
curl -X POST http://localhost:5001/api/article \
  -H "Content-Type: application/json" \
  -d '{"url": "https://mp.weixin.qq.com/s/xxxxxx"}'
```

返回：

```json
{
  "title": "文章标题",
  "content": "<div>正文HTML...</div>",
  "plain_content": "纯文本正文...",
  "images": ["https://mmbiz.qpic.cn/..."],
  "author": "作者名",
  "publish_time": 1741234567
}
```

#### 功能二：公众号名 → 昨天发布的文章

**Step 1 — 搜索公众号，获取 FakeID**

```bash
curl "http://localhost:5001/api/public/searchbiz?query=人民日报"
```

返回示例：

```json
{
  "success": true,
  "list": [{
    "fakeid": "MzA1MjM1ODk2MA==",
    "nickname": "人民日报",
    "alias": "rmrbkwb"
  }]
}
```

**Step 2 — 获取该公众号文章列表**

```bash
curl "http://localhost:5001/api/public/articles?fakeid=MzA1MjM1ODk2MA==&count=100"
```

**Step 3 — 用 Python 按日期过滤昨天文章**

```python
import requests, datetime

# 获取昨天的时间范围
today = datetime.date.today()
yesterday = today - datetime.timedelta(days=1)
yesterday_start = int(datetime.datetime.combine(yesterday, datetime.time(0, 0)).timestamp())
yesterday_end = int(datetime.datetime.combine(today, datetime.time(0, 0)).timestamp())

# 获取文章列表
resp = requests.get(
    "http://localhost:5001/api/public/articles",
    params={"fakeid": "MzA1MjM1ODk2MA==", "count": 100}
)
articles = resp.json().get("app_msg_list", [])

# 过滤昨天发布的
yesterday_articles = [
    a for a in articles
    if yesterday_start <= a["update_time"] < yesterday_end
]

for a in yesterday_articles:
    print(f"[{datetime.date.fromtimestamp(a['update_time'])}] {a['title']}")
    print(f"  {a['link']}")
```

## 完整接口一览

| 方法 | 路径 | 说明 |
|------|------|------|
| `POST` | `/api/article` | 获取文章正文（HTML/纯文本/图片/作者） |
| `GET` | `/api/public/searchbiz?query=` | 按名称搜索公众号 |
| `GET` | `/api/public/articles?fakeid=` | 获取公众号文章列表 |
| `GET` | `/api/public/accountinfo?fakeid=` | 公众号信息（认证主体、原创文章数） |
| `GET` | `/api/rss/{fakeid}` | 获取该公众号 RSS 源 |
| `POST` | `/api/rss/subscribe` | 订阅公众号 |
| `GET` | `/api/health` | 健康检查 |
| `GET` | `/api/admin/status` | 查看登录状态和过期时间 |
| `GET` | `/api/stats` | 限频统计 |

完整文档：**http://localhost:5001/api/docs**

## 给同事使用的方式

### 共用一套服务（推荐）

你部署一套，同事通过 API 调用：
- 同事无需安装任何东西，直接 `curl` 或在自己的程序里调接口
- 所有人共用你的微信账号登录态
- 通过 `/api/stats` 可查看各 IP 的调用量

### 各自部署独立实例

每个同事 clone 项目，在自己机器上启动：
- 各自需要扫码登录（独立微信账号权限）
- 互不影响，但需要各自维护登录态

### 部署到服务器供局域网访问

```bash
# 编辑 .env，设置对外地址
SITE_URL=http://你的服务器IP:5001

# 启动
docker-compose up -d

# 同事直接用 http://服务器IP:5001 访问
```

## 常见问题

**提示"服务器未登录"**

访问 http://localhost:5001/static/login.html 扫码登录。

**触发微信验证**

降低请求频率，等待 30 分钟重试。生产环境建议配置 `PROXY_URLS`（SOCKS5 代理池），避免同一 IP 高频请求。

**凭证 4 天过期**

过期后重新扫码即可。配置 `WEBHOOK_URL`（企业微信机器人）可收到过期提醒。

**RSS 阅读器接入**

打开 http://localhost:5001/static/rss.html，搜索公众号并订阅，复制 RSS 地址添加到阅读器（Feedly、FreshRSS 等均支持）。
